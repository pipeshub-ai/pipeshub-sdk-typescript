# Citation

A populated citation document. Represents a single chunk of source
content (e.g. a passage from a document or record) referenced by an
AI response, together with its provenance metadata.


## Example Usage

```typescript
import { Citation } from "@pipeshub-ai/sdk/models";

let value: Citation = {
  id: "<value>",
  content: "<value>",
  chunkIndex: 901773,
  citationType: "<value>",
  metadata: {
    orgId: "<id>",
    mimeType: "<value>",
    recordId: "<id>",
    recordName: "<value>",
    origin: "<value>",
  },
  createdAt: new Date("2024-11-06T08:44:45.047Z"),
  updatedAt: new Date("2024-01-21T00:37:11.141Z"),
};
```
