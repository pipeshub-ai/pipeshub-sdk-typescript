# ConversationListItem

Conversation summary returned by list endpoints. Identical to
`Conversation` but omits `messages` to keep list payloads small.
Fetch a single conversation to retrieve its messages.


## Example Usage

```typescript
import { ConversationListItem } from "@pipeshub-ai/sdk/models";

let value: ConversationListItem = {};
```
