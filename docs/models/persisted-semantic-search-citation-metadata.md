# PersistedSemanticSearchCitationMetadata

Citation metadata as persisted in MongoDB. Required fields mirror the
Mongoose schema's `required: true` flags; the rest are optional and
may come through as `null` because the AI retrieval service emits
explicit nulls for absent fields.


## Example Usage

```typescript
import { PersistedSemanticSearchCitationMetadata } from "@pipeshub-ai/sdk/models";

let value: PersistedSemanticSearchCitationMetadata = {
  orgId: "<id>",
  mimeType: "<value>",
  recordId: "<id>",
  recordName: "<value>",
  origin: "<value>",
};
```
