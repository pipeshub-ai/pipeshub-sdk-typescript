# RegenerateRequest

Request body for regenerating an AI response. All fields are optional;
when omitted the model and context originally used for the message are
reused. Provide `modelKey` (and the matching display fields) to switch
models, or `filters` to narrow the knowledge sources considered during
regeneration.


## Example Usage

```typescript
import { RegenerateRequest } from "@pipeshub-ai/sdk/models";

let value: RegenerateRequest = {
  modelKey: "05438a37-68f2-4641-a8dc-6c47e63278ca",
  modelName: "gpt-5.4-mini",
  modelFriendlyName: "mini",
  chatMode: "internal_search",
  timezone: "Asia/Calcutta",
  currentTime: new Date("2026-05-11T15:43:21+05:30"),
  tools: [
    "jira.create_issue",
    "confluence.search_content",
  ],
};
```
